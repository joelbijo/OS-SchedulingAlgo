# FCFS Scheduling in C

This project implements First Come First Serve (FCFS) scheduling algorithm.
User inputs number of processes, Arrival Time(AT), and Burst Time(BT) for each process.

## Features
- Calculates Completion Time(CT), Turnaround Time(TAT), Waiting Time(WT)
- Displays Gantt Chart

## How to Run
```bash
gcc fcfs.c -o fcfs
./fcfs
```
